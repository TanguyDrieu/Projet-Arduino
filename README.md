# Projet-Arduino

Voici le Projet Arduino fait sur la période septembre-octobre 2020 au CESI de Rouen pendant notre deuxième année. Notre équipe était composée de : 
Tanguy Drieu, Victorien Goudeau, Antoine Martorell, Arnaud Maturel.

A l'origine, ce projet était fait en C mais pour être exporté sur une carte Arduino, il a dû être transformé en C++, il n'y avait pratiquement rien à modifier.

L'objectif du projet était de faire une station météo embarquée, la carte Arduino et tous ses composants additionnels connectés sont capable d'écrire sur une carte SD 
des données de température, pression, hygrométrie et luminosité. Le fonctionnement peut changer selon 4 modes pouvant être sélectionnés via deux boutons.

Une image du système est disponible dans "Issues".
